# blockchain.js
A Blockchain running on Node.js

## Scripts

| Script     |                                           |
|----------- |-------------------------------------------|
|`build.sh`  | Generate the Docker image                 |
|`serve.sh`  | Run the Docker container with the Web app |
|`test.sh`   | Run the Mocha tests                       |
|`ssh.sh`    | Start a container and run a bash inside it|
